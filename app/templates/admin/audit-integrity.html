Audit Log Integrity Verification

Verify Hash Chain Integrity

This tool verifies that the SHA-256 hash chain of audit log entries is intact. Any modification to a logged entry will break the chain and be detected.

How It Works

1. Hash Chain: Each audit log entry includes a SHA-256 hash computed from the previous entry's hash, user ID, action, resource, and timestamp.

2. Tamper Detection: If any entry is modified, its hash changes, breaking the chain for all subsequent entries.

3. Verification: This tool recomputes each hash and verifies the chain linkage for the selected date range.

4. Retention: Audit logs are retained for 6 years per HIPAA requirements. Use the cleanup tool in the main audit page to remove logs older than the retention period.
