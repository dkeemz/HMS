**app/services/insurance.py** — InsuranceService with state machine (per D-24):

class InsuranceService:
    """Insurance policy lifecycle management."""

    VALID_TRANSITIONS = {
        "pending": ["verified", "expired"],
        "verified": ["active", "expired"],
        "active": ["expired"],
        "expired": [],  # Terminal state
    }

    @staticmethod
    async def create_policy(db, patient_id, data: InsurancePolicyCreate, created_by) -> InsurancePolicy:
        - Create InsurancePolicy with status='pending'
        - db.add, flush, audit log
        - Return policy

    @staticmethod
    async def get_policies_for_patient(db, patient_id) -> list[InsurancePolicy]:
        - select where patient_id, order by created_at.desc()

    @staticmethod
    async def get_policy(db, policy_id) -> InsurancePolicy | None:

    @staticmethod
    async def transition_status(db, policy_id, new_status, changed_by, reason=None) -> InsurancePolicy:
        - Get policy, validate transition against VALID_TRANSITIONS
        - Update status, set timestamp (verified_at, activated_at, expired_at) based on new_status
        - If new_status=='verified': set verified_by=changed_by, verified_at=now
        - If new_status=='active': set activated_at=now
        - If new_status=='expired': set expired_at=now
        - Audit log with old_status, new_status, reason
        - flush, return policy
        - Raise ValueError for invalid transitions

    @staticmethod
    async def get_expired_policies(db) -> list[InsurancePolicy]:
        - select where status != 'expired' AND end_date < today
        - Used for D-33 expired policy warnings

    @staticmethod
    async def get_providers(db) -> list[InsuranceProvider]:
        - select all active providers

router = APIRouter(prefix="/patients/{patient_id}/insurance", tags=["insurance"])

Endpoints:
1. POST /patients/{patient_id}/insurance (status_code=201, response_model=InsurancePolicyResponse)
   - require_permission("patient", "update") — D-32: reception/billing staff manage
   - body: InsurancePolicyCreate
   - Call InsuranceService.create_policy(db, patient_id, body, current_user.id)
   - await db.commit()

2. GET /patients/{patient_id}/insurance (response_model=list[InsurancePolicyResponse])
   - require_permission("patient", "read")
   - Return InsuranceService.get_policies_for_patient(db, patient_id)

3. GET /patients/{patient_id}/insurance/{policy_id} (response_model=InsurancePolicyResponse)
   - require_permission("patient", "read")
   - 404 if not found

4. POST /patients/{patient_id}/insurance/{policy_id}/transition (status_code=200, response_model=InsurancePolicyResponse)
   - require_permission("patient", "update")
   - body: InsuranceStatusTransition (new_status, reason)
   - Call InsuranceService.transition_status(db, policy_id, body.new_status, current_user.id, body.reason)
   - Catch ValueError -> HTTPException(409)

5. GET /patients/{patient_id}/insurance/expired (response_model=list[InsurancePolicyResponse])
   - require_permission("patient", "read")
   - Return policies where status != 'expired' and end_date < today (D-33)

6. GET /insurance/providers (response_model=list[InsuranceProviderResponse])
   - require_permission("patient", "read")
   - Return InsuranceService.get_providers(db)

7. POST /patients/{patient_id}/insurance/{policy_id}/upload-card (status_code=200)
   - require_permission("patient", "update")
   - Accept file upload (UploadFile), save to uploads/insurance/{uuid}.{ext}
   - Update card_image_url on policy
   - D-31: card image upload
